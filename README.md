# 🧁 Bot de WhatsApp - Pastelería Finality

Este es un bot automatizado para la venta de rebanadas de postres artesanales, desarrollado con Node.js, Express y Twilio.

## ✨ Características
- Saludo inicial amigable y personalizado.
- Menú interactivo con fotos de las rebanadas.
- Selección de cantidades y cálculo automático del total.
- Gestión de pedidos (Dirección y métodos de pago).
- Soporta pago por transferencia (envío de comprobante) o efectivo.

## 🚀 Instalación
1. Clona el repositorio.
2. Ejecuta npm install para instalar las dependencias.
3. Configura tus credenciales de Twilio en un archivo .env.
4. Inicia el bot con node app.js.
